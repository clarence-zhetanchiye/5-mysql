符合      含义              说明
+       加法运算       （无法用于字符串的相加！）
-       减法运算
*       乘法运算
/       除法运算       （与 / 运算符类似的是 DIV ，但DIV不能像 / 进行类似这样 a / b / c 的连续除法）
%       取余运算       （与 % 运算符类似的是 MOD ，但MOD不能像 % 进行类似这样 a % b % c 的连续取余）


用途：可用于查询语句中，对查询到的数据表的两列的值进行运算后，展示为查询结果。
